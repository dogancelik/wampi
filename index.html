Wampi is here!

Wampi is a plugin-based WAMP installer and manager. It lets you download different versions of Apache, PHP and MySQL.

You can even download MariaDB and PhpMyAdmin!

Download Github
Requirements: .NET Framework 4.5

Features

It's all plugins!

Wampi is plugin-based that means you can create your own plugins and customize Wampi even more!

Current set of plugins are: Apache, MySQL, PHP, MariaDB and PhpMyAdmin. The bundled plugins will let you download, install and manage your programs.

It's open source!

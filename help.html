What should I download?

There are some abbreviations that might confuse you, here are explanations for them.

  • VC11 = Visual Studio 2012
  • VC10 = Visual Studio 2010
  • VC9 = Visual Studio 2008
  • TS = Thread safe (download this if you want to use PHP with Apache)
  • NTS = Non Thread Safe
